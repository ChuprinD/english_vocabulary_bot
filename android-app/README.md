# Vocab Check — Android fact-check app

Мини-приложение для проверки русских переводов словаря (свайп как в Tinder).

## Возможности

1. **Проверка** — карточка со словом и переводами  
   - свайп **вправо** / кнопка OK — перевод верный  
   - свайп **влево** / кнопка «Правка» — слово уходит на ручную правку  
   - **Swap** — меняет местами основной перевод и первый дополнительный
2. **Правки** — ручной ввод основного и доп. переводов (можно добавлять поля)
3. **Экспорт** — кнопка в шапке активна, когда все слова в статусе OK; делится `vocab_checked.json`

Прогресс сохраняется локально на устройстве.

## Сборка APK

Нужны JDK 17+ и Android SDK (или Android Studio).

```bash
cd android-app
./gradlew.bat assembleDebug
```

APK появится здесь:

`android-app/app/build/outputs/apk/debug/app-debug.apk`

## Установка на Samsung S25 Ultra

1. Скопируй APK на телефон (USB, Telegram, Drive…)
2. Настройки → Безопасность → разреши установку из этого источника
3. Открой APK и установи

## Данные

Словарь берётся из `app/src/main/assets/words_slim.json`  
(собрать заново: `python scripts/make_slim_words.py` из корня репозитория).
